**MiniPokemon** adalah aplikasi game desktop sederhana berbasis **Java** yang dikembangkan untuk mengimplementasikan konsep **Pemrograman Berorientasi Objek (OOP)**. Proyek ini mensimulasikan mekanisme dasar permainan Pokemon dan dikembangkan sebagai bagian dari tugas mata kuliah **Pemrograman Berorientasi Objek (PBO)**.
